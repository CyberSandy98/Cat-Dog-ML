This project distinguishes between Cats and Dogs from an image using Machine Learning.
I have used Object Detection module of Tensorflow to detect cats/dogs from images.
I have used Faster RCNN Inception V2 to train the model.

Now the TFRecords can be generated by typing:

python3 tfrecord.py --csv_input=images/train_labels.csv --image_dir=images/train --output_path=train.record
python3 tfrecord.py --csv_input=images/test_labels.csv --image_dir=images/test --output_path=test.record

Training Model:

python3 model_main.py --logtostderr --model_dir=training/ --pipeline_config_path=training/faster_rcnn_inception_v2_pets.config
(model_main.py is located inside object_detection folder)

Exporting inference graph :

python3 export_inference_graph.py --input_type image_tensor --pipeline_config_path training/faster_rcnn_inception_v2_pets.config --trained_checkpoint_prefix training/model.ckpt-XXXX --output_directory inference_graph

XXXX is the highest value of model.ckpt

Testing Model:
 
python3 catdog.py
(catdog.py is located inside object_detection folder)

Transform images :

python3 transformimageres.py -d images/ -s 800 600
